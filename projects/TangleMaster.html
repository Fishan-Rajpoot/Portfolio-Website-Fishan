Tangle Master

Game Description

Tangle Master is a Unity-based puzzle game where players must untangle intricate patterns of ropes and nodes. The game combines intuitive mechanics with vibrant visuals, challenging players' problem-solving skills across increasingly complex levels. Developed with C#, it ensures smooth interactions and engaging gameplay.

My Role

I was responsible for designing the puzzle mechanics and implementing the core gameplay using Unity and C#. Additionally, I created the visual effects and optimized the game for performance across different devices.
