Home Hammer 2D

Game Description

Home Hammer 2D is a 2D Unity game where players renovate homes through creative puzzles. The game combines engaging gameplay with charming visuals, allowing players to design and restore houses. Built with C#, it offers smooth interactions and intuitive controls for a delightful experience.

My Role

I developed the core puzzle mechanics and renovation systems using Unity and C#. I also designed the 2D art assets and ensured the game was optimized for a seamless player experience.
