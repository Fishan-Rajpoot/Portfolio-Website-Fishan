Home Hammer 2D

Game Description

Step into the relaxing world of Home Hammer 2D, a cozy and satisfying 2D game built with the Unity engine. A once-beautiful family home now lies in ruins—and it's up to you to bring it back to life.

Pick up your tools and experience the calming satisfaction of ASMR-inspired tasks like hammering, painting, fixing furniture, and more. Each tap and action is designed to immerse you in a peaceful, feel-good repair experience.

My Role

In Home Hammer 2D, I served as the Unity Developer responsible for bringing the entire game to life using the Unity engine and my programming skills. I collaborated closely with a multidisciplinary team:

  • Worked with the 2D Artist to implement all character and environment sprites.
  • Integrated UI assets provided by the UI Artist, ensuring smooth and responsive interactions.
  • Added and synced sound effects and music provided by the Sound Artist to enhance the ASMR experience.
  • Translated the core gameplay concept provided by the Game Designer into functional game mechanics.

From coding the core systems and level progression to handling animations, UI transitions, and polish, I was responsible for the full implementation of the project in Unity.
